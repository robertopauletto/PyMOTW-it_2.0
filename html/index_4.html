Il Modulo Python della Settimana

Questo sito (in continuo aggiornamento) contiene la traduzione italiana dell'apprezzata serie Python Module of the Week (PyMOTW), curata da Doug Hellmann.

Questa serie di articoli offre una analisi della libreria standard Python attraverso vari esempi e spiegazioni.

Come neofita del linguaggio Python ho molto apprezzato gli articoli di Doug. che si sono rivelati, almeno per me, una ottima fonte di apprendimento.

Spero che questa traduzione possa aiutare coloro che programmano in Python e non hanno molta dimestichezza con la lingua inglese.
